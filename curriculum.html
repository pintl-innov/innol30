교육 구성 원칙

주차별 커리큘럼

주차 학습 목표 핵심 활동 성과물
1주차
기초 다지기
  • 디지털 전환 이해
  • 데이터 정리 기본 역량 확보
  • 프로젝트 킥오프 & 요구사항 워크숍
  • 구글 스프레드시트/에어테이블로 데이터 구조 설계
  • 작업일지 표준 입력 양식 만들기
  • 표준화된 작업일지 양식 v1
  • 데이터 품질 체크리스트
2주차
RAG 기초
  • LLM & RAG 개념 이해
  • 문서 전처리 & 임베딩 경험
  • 작업일지 텍스트 클리닝 및 분류 실습
  • 벡터DB(예: Pinecone/Weaviate) 개념 소개 및 데모
  • 노코드 RAG 툴 활용해 Q&A 봇 프로토타입 만들기
  • 작업일지 기반 RAG Q&A 시나리오 초안
  • FAQ 데이터셋 및 임베딩 파일
3주차
n8n 자동화
  • n8n 환경 이해 및 워크플로 설계
  • 데이터 파이프라인 자동화
  • n8n 기초 노드(Trigger, Function, HTTP, Spreadsheet) 실습
  • RAG 결과와 대시보드 데이터 연동
  • 알림/보고 자동화 플로우 구축
  • n8n 기반 데이터 수집 & 알림 워크플로
  • 자동화 운영 가이드 초안
4주차
대시보드 & 프로젝트 완성
  • 시각화 도구 실습
  • 프로젝트 통합 및 발표
  • 로코드 BI 도구(예: Google Looker Studio, Metabase) 실습
  • 성과 지표 정의 및 대시보드 제작
  • 최종 통합 리허설 & 피드백
  • 최종 통합 대시보드
  • RAG + n8n 연동 데모
  • 운영 매뉴얼 & 유지보수 계획

주간 세부 일정 (총 12회 오프라인 세션)

회차 주제 주요 내용 과제
1회차 프로젝트 오리엔테이션 목표 공유, 데이터 수집 현황 분석, 페르소나 정의 팀별 역할 분담 & 개선 아이디어 도출
2회차 데이터 구조화 워크숍 데이터 항목 표준화, 오류 유형 파악 작업일지 양식 작성 및 데이터 정비
3회차 데이터 품질 진단 검증 규칙 수립, n8n 없이 검증 체크리스트 작성 품질 진단 리포트 제출
4회차 LLM & RAG 이해 프롬프트 기초, 임베딩 개념, RAG 아키텍처 소개 작업일지 FAQ 초안 작성
5회차 문서 전처리 실습 텍스트 정제, 태깅, 메타데이터 부여 정제된 문서 세트 업로드
6회차 RAG 프로토타이핑 벡터DB 데모, 노코드 RAG 툴 구성 RAG Q&A 시나리오 테스트
7회차 n8n 기초 인터페이스 소개, 워크플로 만들기 체험 개별 워크플로 과제 설계
8회차 데이터 파이프라인 API 연동, 스프레드시트 자동 업데이트 데이터 수집 플로 완성
9회차 알림 & 보고 자동화 슬랙/이메일 알림, 주간 보고 자동 생성 알림 플로 테스트 로그 제출
10회차 대시보드 기초 시각화 원칙, 주요 KPI 설계 KPI 정의서 초안 작성
11회차 대시보드 제작 Looker Studio/Metabase 실습, 필터/차트 구성 대시보드 초안 제출
12회차 최종 통합 & 발표 RAG + n8n + 대시보드 연결, 운영 계획 정리 최종 발표 & 피드백 반영 계획

지원 프로그램

  • 주차별 온라인 Q&A 문답지 운영 (슬랙/노션 활용)
  • 멘토 1:1 코칭 2회 제공 (2주차, 4주차)
  • 평가 루브릭 제공 및 프로젝트 품질 진단
  • 교육 종료 후 4주간 운영 지원 헬프데스크 운영
